# 지역 인구 현황 및 소멸 위험 분석 시스템 기술 보고서

## 1. 개요

본 시스템은 행정안전부 주민등록 인구 통계 데이터를 활용하여 대한민국 각 지역의 인구 구조를 분석하기 위해 개발되었습니다. 기존의 가상 데이터를 사용하던 방식에서 벗어나, 공공데이터 포털의 실시간 데이터를 수집 및 분석함으로써 실제 사회 현상을 반영한 결과를 도출하는 것을 목적으로 합니다.

## 2. 주요 기능 및 요구사항 구현 현황

본 프로그램은 소프트웨어 요구사항 명세서(SRS)에 정의된 기능적 및 비기능적 요구사항을 다음과 같이 구현하였습니다.

### 2.1 데이터 수집 및 전처리

* **외부 데이터 수집 (FR-01)**: `requests` 라이브러리를 사용하여 지정된 오픈 API에 GET 요청을 전송하고 데이터를 수신합니다.


* **서버 응답 처리**: HTTP 상태 코드가 200(성공)인 경우 데이터를 Pandas DataFrame으로 변환하며, 에러 발생 시 사용자에게 통신 실패 메시지를 출력합니다.


* **데이터 정제 (FR-02)**: 원본 데이터에 포함된 콤마(,) 등 숫자 문자열을 정수형(Int)으로 변환하여 수치 계산이 가능하도록 정제합니다.


* **지표 산출**: 전체 인구 대비 65세 이상 인구의 백분율인 '고령화 비율'과 가임 여성인구 대비 고령 인구 비율인 '소멸위험지수'를 계산합니다.



### 2.2 시각화 및 리포트 생성 (FR-03)

* **이중 축 그래프 생성**: `matplotlib`의 `twinx()` 함수를 활용하여 좌측 Y축에는 총 인구수를, 우측 Y축에는 고령화 비율을 배치한 차트를 생성합니다.


* **지리 정보 시각화**: `folium` 라이브러리를 통해 각 지역별 위경도 좌표에 분석 지표를 매핑하고, 소멸 위험 단계에 따라 동적 마커가 표시된 인터랙티브 지도를 생성합니다.
* **결과 저장**: 분석 결과는 `result_YYYYMMDD.xlsx` 형식의 엑셀 파일로 자동 저장되어 데이터 보존 및 보고서 작성을 지원합니다.



## 3. 시스템 설계 전략 및 예외 처리

### 3.1 API 보안 설계

본 시스템은 보안 표준 및 관리 효율성을 고려하여 다음과 같이 설계되었습니다.

* **인증키 관리**: API 인증키(Service Key)는 사용자별 고유 정보이므로 소스 코드 내에 하드코딩하지 않고 Placeholder(`YOUR_API_KEY`)를 사용하여 설계하였습니다. 이는 인증 정보 유출(Credential Leakage)을 방지하고 개별 할당량을 보호하기 위함입니다.
* **Stateless 구조**: 프로그램 실행 시 환경에 따라 인증키를 유연하게 적용할 수 있는 구조를 지향합니다.

### 3.2 결함 허용 설계 (NFR-01)

네트워크 불안정 또는 서버 접속 불가 상황에 대비한 **Fallback Mechanism**을 구축하였습니다.

* 인터넷 연결이 단절되거나 서버 에러가 발생할 경우 프로그램이 중단되지 않고 사용자에게 경고 메시지를 표시합니다.


* 사용자의 선택에 따라 로컬에 내장된 샘플 데이터셋을 로드하여 분석 프로세스를 지속함으로써 시스템의 가용성을 확보하였습니다.



## 4. 설치 및 실행 방법

### 4.1 필수 라이브러리

본 프로그램 구동을 위해 다음 라이브러리의 설치가 필요합니다.

```bash
pip install pandas requests matplotlib folium openpyxl

```

### 4.2 실행 프로세스

1. 명령 프롬프트 또는 터미널에서 파이썬 파일을 실행합니다.
2. 데이터 서버 접속 시도 후, 통신 상태에 따라 실시간 데이터 또는 샘플 데이터를 선택합니다.
3. 데이터 처리 완료 후 자동으로 시각화 창이 나타나며, 폴더 내에 결과 파일이 생성됩니다.

## 5. 최종 산출물 리스트

* **데이터 리포트**: `result_YYYYMMDD.xlsx` (정제된 인구 통계 데이터셋) 
* **공간 분석 리포트**: `population_analysis_map.html` (지역별 위험 지수 매핑 지도)
* **정적 리포트**: 데이터 기준 연월이 포함된 시각화 그래프 창
